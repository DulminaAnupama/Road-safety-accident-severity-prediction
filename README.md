# 🚗 Road Safety Insights: Predicting Accident Severity

## 📌 Project Overview
This project uses data mining and machine learning techniques to predict accident severity based on historical road accident data.

## 🎯 Objectives
- Analyze accident data to identify key risk factors
- Build predictive models for accident severity
- Visualize patterns and trends

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## 📊 Key Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine learning model building
- Performance evaluation

## 📈 Results
- Achieved accurate predictions of accident severity
- Identified key contributing factors (e.g., weather, location)
